Medicine OCR API

Takes a medicine label image, runs Azure Computer Vision Read OCR, and returns
one selected text result.

## How it works

1. The uploaded image is sent to Azure Read API (`vision/v3.2/read/analyze`).
2. Each OCR line is cleaned by removing numbers and dosage/package words such
   as `Щ‚Ш±Шµ`, `Щ…Ш¬Щ…`, `tablets`, `mg`, and similar terms.
3. Lines that are only numbers or dosage/package words are ignored.
4. If any cleaned Arabic medicine-name line has confidence >= `0.7`, the
   biggest Arabic box is returned.
5. If no Arabic candidate passes that threshold, the biggest English
   medicine-name box is returned.
6. Duplicate words inside the selected name are removed.

## Configuration

Set these values in `.env`:

```bash
AZURE_OCR_ENDPOINT=https://your-resource.cognitiveservices.azure.com/
AZURE_OCR_KEY=your_subscription_key
```

Optional settings:

```bash
AZURE_OCR_TIMEOUT=30
AZURE_OCR_MAX_POLLS=60
AZURE_OCR_POLL_SECONDS=1
ARABIC_CONFIDENCE_THRESHOLD=0.7
```

# Medicine Name OCR API

A **FastAPI-based Computer Vision API** for extracting **medicine names from images** using a hybrid pipeline combining **Image Processing, YOLO Object Detection, and EasyOCR**.

The system detects the **region containing the medicine name**, crops it, and then applies OCR to extract the text.

---

# Features

- Image preprocessing for better OCR accuracy
- YOLO-based medicine name region detection
- Automatic bounding box cropping
- EasyOCR text extraction
- Clean modular FastAPI architecture

# OCR Pipeline

The system extracts medicine names through **four main stages**.

---

## 1. Image Preprocessing (Sharpening)

Before detection, the image is enhanced using **image sharpening** to make the text clearer.

Benefits:

- Improves edges
- Enhances blurry text
- Improves OCR accuracy

Example kernel used:

```
[ 0 -1 0
-1 5 -1
0 -1 0 ]
```

---

## 2. YOLO Object Detection

A trained **YOLO model (`best.pt`)** is used to detect the **medicine name region**.

The model:

1. Takes the sharpened image
2. Predicts bounding boxes
3. Returns coordinates of the detected region

Example:

`x1, y1, x2, y2`

---

## 3. Cropping the Detected Region

The detected bounding box is used to **crop the medicine name area**.
This step removes unnecessary background and improves OCR performance.

```
Original Image

в–ј
Detected Bounding Box
в”‚

```

---

## 4. OCR using EasyOCR

Finally, **EasyOCR** extracts the text from the cropped region.

###Output includes:

```
- Medicine name
- Confidence score
```
